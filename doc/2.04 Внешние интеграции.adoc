=== Внешние интеграции

1. Keycloak в качестве IAM сервиса, с выделенным realm - teamBridge, который импортирует группы и учетные записи пользователей компании
2. Active Directory или другой LDAP каталог, как внешний источник учетных записей сотрудников компании
3. Push Notification сторонние сервисы: APN (Apple Push Notifications), FCM (firebase cloud messaging)


|===
|Outbounding |Inbounding |Protocol |Description
| Keycloak | AD/LDAP | LDAP (LDAPS) | Синхронизация корпоративных учетных записей и групп через federation
| ApiGateway | Keycloak | REST: HTTPS | Проверка и обработка аутентификации HTTP запросов
| WebSocketGateway | Keycloak | REST: HTTPS | Проверка JWT токена (по openid configuration/JWKS) при установке WS соединения
| ChatService | Keycloak | REST: HTTPS | Чтение профиля сотрудника и статуса учетной записи (активна/заблокирована)
| NotificationService| APN/FCM |  REST: HTTPS | Отправка уведомлений на устройство пользователя
|===
